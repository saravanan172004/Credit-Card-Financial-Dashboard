# 💳 Credit Card Financial Dashboard

## 📊 Project Overview

This project is a **Power BI Credit Card Financial Dashboard** designed to analyze customer transactions, spending patterns, and revenue insights.
It provides a **data-driven view of credit card usage**, helping stakeholders make informed financial decisions.

## 🎯 Objective

The main goal of this project is to:

* Analyze credit card transactions and customer behavior
* Track key performance metrics (KPIs)
* Generate actionable insights using interactive dashboards
* Support business decision-making with data visualization

---

## 🛠️ Tools & Technologies Used

* **Power BI** – Data visualization & dashboard creation
* **SQL / MySQL** – Data storage and querying
* **Excel / CSV** – Data preprocessing
* **DAX (Data Analysis Expressions)** – Calculations and measures

## 🔄 Workflow

1. Data collection from CSV/Excel files
2. Data cleaning and transformation using Power Query
3. Import data into SQL database
4. Connect SQL database with Power BI
5. Create relationships and data model
6. Build DAX measures for KPIs
7. Design interactive dashboards
8. Generate insights
